📊 Sales Data Analysis – Week 3
📌 Project Overview

This project introduces data analysis using Python and Pandas by working with a real-world sales dataset. The objective is to load, clean, analyze, and generate insights from data.

It demonstrates how raw data can be transformed into useful business information.

🎯 Objectives
Understand the basics of data analysis
Work with CSV files using Pandas
Clean and preprocess data
Perform statistical analysis
Generate meaningful insights
